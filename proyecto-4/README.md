# Proyecto 11 — Predicción de reservas de petróleo (OilyGiant)

## Descripción general
La compañía **OilyGiant** planea abrir 200 nuevos pozos de petróleo.  
El objetivo principal de este proyecto es **encontrar las mejores ubicaciones** para construirlos, de modo que la empresa obtenga el **mayor beneficio posible** y minimice los riesgos de pérdida.

## Estructura de archivos
proyecto-4/
├── data/ # Archivos CSV con datos geológicos
├── notebooks/ # Jupyter Notebook principal
├── README.md # Descripción del proyecto
└── requirements.txt # Librerías necesarias

## Librerías usadas
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn

## Resultado esperado
Entrenar un modelo de regresión lineal y, con bootstrapping, evaluar el riesgo/beneficio para recomendar la región óptima donde abrir pozos.
